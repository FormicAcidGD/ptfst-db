# ptfst-db

## What is PTFST-DB?

ptfst-db is a package that contains data about the ATC24 server for PTFS, used by all PTFS Tools Applications

## Installation

### NPM

```sh
npm install ptfst-db
```

### Yarn

```sh
yarn add ptfst-db
```
